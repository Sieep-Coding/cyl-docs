Getting Started

Initial Steps

To begin using Check Your Lua, follow these steps to set up your environment and run your first test. This guide provides an overview of the process, ensuring you can dive into testing your Lua code without hassle.

Important: Ensure you have Lua installed on your machine to run the tests effectively.

Install the App

Follow these steps to install Check Your Lua:

  1. Clone the repository from GitHub to your local machine.

  2. Navigate to the project directory in your terminal.

  3. Run the setup script to install all necessary components.

Opening the App

Once installed, you can open Check Your Lua and start testing your Lua scripts:

  1. Open your terminal and navigate to the app directory.

  2. Run the command to start the app.

  3. Follow the on-screen prompts to set up your first test.

  4. Begin testing your Lua scripts and enjoy the features!

Check Your Lua offers features like colored output, zero dependencies, and an intuitive interface, making it the perfect tool for Lua developers.
